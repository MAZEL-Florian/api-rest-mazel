# Projet API rest

## Commande à exécuter

```
docker-compose up
```

Pour accéder au projet, vous devez faire une requête Postman pour ajouter un admin de la manière suivante :

![Screenshot postman image](/screenshot-postman.png "screenshot postman image")

Si vous ne le faites pas, vous n'aurez aucun moyen de vous connecter car il n'existe pas de page d'inscription.
